# Simple Web Backend API

This is a basic, easy-to-understand backend project created using Node.js and MongoDB. It acts as the "brain" and data storage behind a simple web application.

## What Does This Project Do?
This code demonstrates three fundamental, yet very important, features used by almost every major website:
1. **User Accounts (Authentication):** Users can securely sign up and log in using an email and password.
2. **File Uploading:** Allows users to upload their own picture files perfectly and securely.
3. **Payments System:** A simple test route to practice verifying mock money transactions.

## How to Run This on Your Computer

1. Open your terminal or command prompt where the code is saved.
2. Install the necessary files by running:
   ```bash
   npm install
   ```
3. Start the server by running:
   ```bash
   node index.js
   ```
